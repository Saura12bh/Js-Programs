2 Way ->using fetch() method

fetch() is a modern JavaScript API used to make HTTP requests (like AJAX) to load data from a server without reloading the page.
It returns a Promise, so we can use .then() and .catch() to handle the result.
It is a replacement for old AJAX (XMLHttpRequest).

Syntaxt=>

fetch("file_or_url")
  .then(response => response.text()) // or response.json()
  .then(data => {
    // use the data
  })
  .catch(error => {
    // handle error
  });

| Concept           | Explanation                          |
| ----------------- | ------------------------------------ |
| `fetch()`         | Used to send HTTP request            |
| `response.ok`     | Checks if status is 200â€“299          |
| `response.text()` | Converts response body to text       |
| `.json()`         | Use this instead if the file is JSON |
| `.catch()`        | Handles network or response errors   |
